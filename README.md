# Goliaths of Pollution and Davids of the Polluted

### Climate Inequality — Data Story Project

Team project for the MSc Data Science course.

## Structure

The project is organized around **7 research questions** [Project Specification.pdf](https://github.com/tince250/climate-change-ds/blob/main/Project%20Specification.pdf)).

Each research question is implemented in a separate Jupyter notebook and includes:

* Data preprocessing
* Exploratory analysis
* Visualizations
* Data-driven conclusions

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/tince250/climate-change-ds.git
cd climate-change-ds
```

2. Create a virtual environment:

```bash
python -m venv venv
```

3. Activate the environment:

**Mac/Linux**

```bash
source venv/bin/activate
```

**Windows**

```bash
venv\Scripts\activate
```

4. Install dependencies:

```bash
pip install -r requirements.txt
```

5. Then open and run each notebook individually.
